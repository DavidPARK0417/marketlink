# 네비게이션 컴포넌트 명세서

## 개요
애플리케이션의 주요 네비게이션을 담당하는 컴포넌트 모음입니다. 헤더, 사이드바, 네비게이션 아이템으로 구성되며, 반응형 디자인을 지원합니다.

---

## 1. Sidebar 컴포넌트

### 설명
애플리케이션 주요 메뉴를 표시하는 사이드바 컴포넌트입니다.

### 레이아웃
- **데스크톱**: 좌측에 고정된 사이드바 (너비: 240px)
- **모바일**: 숨김 상태 (햄버거 메뉴로 제어)

### Props

| Props | Type | 설명 |
|-------|------|------|
| `isOpen` | `boolean` | 모바일에서 사이드바 오픈 상태 |
| `items` | `NavItem[]` | 네비게이션 아이템 배열 |
| `onItemClick` | `(path: string) => void` | 아이템 클릭 핸들러 |
| `className` | `string` | (선택) 커스텀 CSS 클래스 |

### 스타일
- **배경색**: `#FFFFFF` (흰색)
- **테두리**: 우측에 `1px solid #E5E7EB` (회색)
- **스크롤**: 항목이 많을 경우 Y축 스크롤 활성화
- **z-index**: 모바일에서 40 (오버레이 위)

### 반응형 동작
| 기기 | 동작 |
|-----|------|
| 데스크톱 (≥768px) | 항상 표시, 고정 위치 |
| 태블릿 (640px-767px) | 좌측 슬라이드 오버레이 (오버레이 포함) |
| 모바일 (<640px) | 좌측 슬라이드 오버레이 (오버레이 포함) |

## 2. Header 컴포넌트

### 설명
애플리케이션 상단 헤더로 검색, 사용자 메뉴, 햄버거 버튼을 포함합니다.

### Props

| Props | Type | 설명 |
|-------|------|------|
| `onMenuToggle` | `() => void` | 햄버거 메뉴 클릭 핸들러 |
| `onSearch` | `(query: string) => void` | 검색 입력 핸들러 |
| `user` | `User` | 사용자 정보 객체 |
| `className` | `string` | (선택) 커스텀 CSS 클래스 |

### User 인터페이스

```typescript
interface User {
  name: string;           // 사용자 이름
  email: string;          // 사용자 이메일
  avatar?: string;        // (선택) 프로필 이미지 URL
}
```

### 레이아웃
- **높이**: 64px
- **배경색**: `#FFFFFF` (흰색)
- **테두리**: 하단에 `1px solid #E5E7EB` (회색)
- **패딩**: 16px (상하좌우)

### 구성 요소

#### 2.1 햄버거 버튼 (모바일만)
- **노출**: 640px 이하에서만 표시
- **크기**: 24x24px
- **아이콘**: 세로 선 3개 (≡)
- **색상**: `#374151` (검은색)
- **호버**: `#10B981` (초록색)

#### 2.2 검색 바
- **노출**: 768px 이상에서만 표시
- **너비**: 300px
- **높이**: 40px
- **배경색**: `#F3F4F6` (밝은 회색)
- **테두리**: `1px solid #D1D5DB` (회색)
- **모서리**: 반경 8px
- **패딩**: 12px
- **플레이스홀더**: "검색..."
- **호버**: 테두리색 `#10B981`

#### 2.3 사용자 메뉴
- **아이콘**: 프로필 이미지 또는 기본 아바타
- **크기**: 36x36px
- **모서리**: 반경 50% (원형)
- **호버**: 테두리 `2px solid #10B981`

## 3. NavItem 컴포넌트

### 설명
사이드바에 표시되는 개별 네비게이션 아이템 컴포넌트입니다.

### Props

| Props | Type | 설명 |
|-------|------|------|
| `label` | `string` | 메뉴 텍스트 |
| `path` | `string` | 라우트 경로 |
| `icon` | `React.ReactNode` | 메뉴 아이콘 |
| `isActive` | `boolean` | 활성 상태 여부 |
| `onClick` | `(path: string) => void` | 클릭 핸들러 |
| `badge` | `number` | (선택) 배지 숫자 (알림 등) |

### 인터페이스

```typescript
interface NavItem {
  label: string;
  path: string;
  icon: React.ReactNode;
  isActive?: boolean;
  badge?: number;
  children?: NavItem[];  // (선택) 서브 메뉴
}
```

### 스타일

#### 기본 상태
- **높이**: 44px
- **패딩**: 12px 16px
- **텍스트 색상**: `#6B7280` (회색)
- **아이콘 색상**: `#6B7280` (회색)
- **배경색**: `transparent`
- **폰트 크기**: 14px

#### 호버 상태
- **배경색**: `#F9FAFB` (매우 밝은 회색)
- **텍스트 색상**: `#374151` (더 어두운 회색)
- **아이콘 색상**: `#374151` (더 어두운 회색)

#### 활성 상태 (isActive = true)
- **좌측 인디케이터**: `4px solid #10B981` (초록색)
- **배경색**: `#F0FDF4` (매우 밝은 초록색)
- **텍스트 색상**: `#10B981` (초록색)
- **아이콘 색상**: `#10B981` (초록색)
- **폰트 가중치**: 600 (볼드)

#### 배지
- **크기**: 18x18px
- **배경색**: `#EF4444` (빨강)
- **텍스트 색상**: `#FFFFFF` (흰색)
- **폰트 크기**: 10px
- **모서리**: 반경 50% (원형)
- **위치**: 우측에 표시

### 레이아웃
```
[아이콘] [레이블 텍스트] ... [배지]
         4px             나머지 공간   8px
```

## 5. 반응형 동작 정리

### 데스크톱 (≥768px)
- 사이드바: 고정 표시
- 햄버거 버튼: 숨김
- 검색 바: 표시
- 레이아웃: 사이드바 + 메인 콘텐츠

### 태블릿 (640px-767px)
- 사이드바: 숨김 (오버레이)
- 햄버거 버튼: 표시
- 검색 바: 표시
- 레이아웃: 풀 너비 메인 콘텐츠

### 모바일 (<640px)
- 사이드바: 숨김 (오버레이)
- 햄버거 버튼: 표시
- 검색 바: 숨김
- 레이아웃: 풀 너비 메인 콘텐츠

## 6. 접근성 (A11y)

- 모든 버튼에 `aria-label` 속성 포함
- 활성 네비게이션 항목에 `aria-current="page"` 추가
- 키보드 네비게이션 지원 (Tab, Enter)
- 충분한 색상 대비 확보 (WCAG AA 표준)

## 7. 애니메이션

- 사이드바 오픈/클로즈: 300ms ease-in-out
- 배경 오버레이 페이드: 300ms ease-in-out
- 호버 효과: 150ms ease-out
- 활성 인디케이터 나타남: 200ms ease-out
