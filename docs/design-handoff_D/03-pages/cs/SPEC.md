# 고객센터/문의 (CS) 디자인 명세서

## 1. 개요
- **화면 ID**: `WH-CS-001`
- **화면명**: 고객센터 (통합)
- **역할**: 상품 문의(판매자향)와 1:1 문의/FAQ(플랫폼향)를 탭으로 구분하여 제공합니다.
- **Route**: `/wholesaler/cs`
- **Query Params**: `?tab=product` (상품문의), `?tab=inquiry` (1:1문의)

## 2. 화면 구성 (탭 구조)

### 2.1 상품문의 탭 (`activeTab === 'product'`)
- **역할**: 소매 파트너(구매자)가 보낸 문의를 확인하고 답변하는 기능.
- **UI 스타일**: `dashboard`, `orders`와 동일한 Green/Yellow 테마.
- **통계 카드**: 3열 (전체, 대기, 완료) - 클릭 시 필터링.
  - 대기: `#fbbf24` (Yellow) Ring/Text.
  - 완료: `#10B981` (Green) Ring/Text.
- **문의 카드 리스트**:
  - 답변 대기 시 '답변하기' 버튼 (`bg-[#10B981]`) 표시.
  - 완료 시 '답변 확인' 버튼 (`bg-gray-100`) 표시.

### 2.2 통합 고객센터 탭 (`activeTab !== 'product'`)
- **역할**: 도매 파트너가 플랫폼(관리자)에게 문의하거나 공지사항을 확인.
- **헤더 배너**:
  - `bg-gradient-to-r from-blue-500 to-blue-600` (Blue Theme)
  - 검색창 포함 (자주 묻는 질문 검색).
- **서브 탭**:
  - 문의내역 (1:1), 자주묻는질문(FAQ), 고객의 소리(VOC), 공지사항, 입점문의.
- **1:1 문의 내역**:
  - 테이블 형태.
  - 상태 뱃지: Blue Theme (접수완료: Gray, 답변완료: Blue).

## 3. 색상 테마 구분 (Context Separation)
이 페이지는 두 가지 성격의 업무가 공존하므로 색상 테마를 의도적으로 분리하였습니다.

| 구분 | 업무 성격 | 테마 색상 |
|---|---|---|
| **상품 문의** | 판매자로서 고객 응대 | **Green (#10B981)** / Yellow (#fbbf24) |
| **고객 센터** | 플랫폼 이용자로서 문의 | **Blue (#2563eb)** / Gray |

## 4. 모달 (Modals)
- **새 문의 작성**: Blue Theme 버튼/인풋 포커스.
- **VOC / 입점 문의**: 각각의 성격에 맞는 아이콘 및 안내 문구 포함.

## 5. 반응형 동작
- **모바일**:
  - 상단 탭 네비게이션: 가로 스크롤 (`overflow-x-auto`).
  - 테이블: 카드 리스트로 변환.
  - 배너 텍스트: 줄바꿈 처리.
